### How to run the script?

1. Install [Python 3](https://www.python.org/)
2. Install requests and Beautiful Soup
    ```
    $ pip3 install requests
    $ pip3 install beautifulsoup4
    ```
3. Run
    ```
    $ python3 simpledesktop-bot.py
    ```
